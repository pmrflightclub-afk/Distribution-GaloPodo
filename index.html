Google — Drive (synchro) & Agenda

Un seul ID client Google (le vôtre) et une seule connexion activent Drive (synchro auto téléphone↔PC) ET l'Agenda. La connexion se fait avec votre compte Google (votre e-mail). Elle reste active ~1 h puis, si l'app est restée fermée longtemps, une reconnexion est demandée (limite Google côté navigateur). Rien n'est partagé avec l'auteur de l'app.

ℹ️ La synchro Drive est en veille (mode fichier actif ci-dessus). L'Agenda reste utilisable ; l'ID client ci-dessous sert aussi à l'Agenda.

❓ Aide détaillée : créer mon ID Google (gratuit, une seule fois)

Google Cloud Console = l'espace développeur gratuit de Google. On y crée un « identifiant » qui autorise cette app à accéder à votre Drive/Agenda (personne d'autre).

💡 Astuce clé : n'utilisez PAS les menus (ils changent). Utilisez la barre de recherche tout en haut de la console et tapez le nom de l'écran (ex. « Google Drive API », « Écran de consentement OAuth », « Identifiants ») pour y aller directement.

1. Ouvrez console.cloud.google.com, connectez-vous.
2. En haut à gauche, cliquez le sélecteur de projet → Nouveau projet → nom « GaloPodo » → Créer, puis sélectionnez-le.
3. Recherchez « Google Drive API » → Activer. Puis « Google Calendar API » → Activer.
4. Recherchez « Écran de consentement OAuth » → type Externe → renseignez le nom de l'app + votre e-mail (assistance et contact) → enregistrez. Dans Utilisateurs test (ou « Audience »), ajoutez votre adresse Gmail.
5. Recherchez « Identifiants » → + Créer des identifiants → ID client OAuth → type Application Web.
6. Dans « Origines JavaScript autorisées » → + Ajouter un URI → collez l'adresse de votre app (barre d'adresse, sans le / final ni le chemin), normalement : https://pmrflightclub-afk.github.io → Créer.
7. Copiez l'ID client (finit par .apps.googleusercontent.com) et collez-le dans le champ ci-dessus.

Le même ID sert à Drive ET Agenda. En mode « test », seul votre compte y a accès (parfait). Ensuite : « Connecter Drive » ici, et/ou « ↻ Google » dans l'onglet Agenda.
